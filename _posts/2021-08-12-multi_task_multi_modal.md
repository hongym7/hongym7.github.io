## Multi Task & Multi Modal
- Single Task : Input X ---> Output Y (ex. Object Classification)
- Multi Task  : Input X ---> Output Y1, Y2, Y3 (ex. Object Detection)
- Multi Modal : Input X1, X2, X3 ---> Output Y (ex. 동영상(음성, 영상) 입력을 통한 예측 )
